# moon-and-myth

Moon and Myth is a project created for ATLS-5519: Design a Sciene Exhibit, a course at the ATLAS Institue at the Univeristy of Colorado Boulder. 

Moon and Myth explores two kinds of science; the one we believed and the one we learned. By examining how we explained Moon phenomemna through history, folklore, and myth we can compare that to modern science and discovery based on ovbservation, data, and the scientific method that we have become accustomed to.
